Twitter Clone Using Ember.js and Node.js

Project Blueprint

The main objective of this close was to learn a new framework. I was very interested in Ember and creating a clone seemed the ideal way of learning the functional capabilities of the framework.

  • Date

    Feb 20th, 2016

User Experience

Ember.js contains a plugin called "liquid fire" allowing for amazing transitions between pages. An advantage of the Framework is incredible responsiveness, the pages seem to snap instantly onto screen. One disadvantage however was the incredibly long load time between server restarts, average time being 18 seconds.
